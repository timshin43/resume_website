Automation Script

Problem:
Data for the Center for Innovation in Teaching and Learning were kept on-line in Canvas LMS. People spent days manually retrieving detailed data from the tool. The basic Canvas`s functionality allows users to download only limited information. As a result, data analysts manually transferred data from Canvas to Excel spreadsheets, that could take days of work.
Solution:
After exploring the issues, I created a python script using Canvas`s API. All data manipulation was done in the Jupiter Notebook. The script retrieved all needed data, processed them and returned the csv file. The script became the base of the program that was used by the whole department. It helped save hundreds of working hours
